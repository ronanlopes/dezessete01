How to install WordPress


This is a short walkthrough of WordPress installation. This part will be useful if you install WordPress theme for the first time.

Before you start, you must have a server that supports PHP and provide access to a MySQL database. First step is to install WordPress on your server. You must download WordPress ZIP file from http://wordpress.org/download/, extract the files and you’ll get a folder named wordpress. Now you must upload all files from this folder to your server via FTP client. Content from this folder must be located in the root of the website, so that this location will be the URL of your website.

If you will access your site URL now, you should see only a note: “There doesn’t seem to be a wp-config.php file…”.

In next step you must manually create a configuration file. Open the wordpress folder and make a copy of file named wp-config-sample.php. Change name of this copy to wp-config.php and open this file in an editor, it can be a simple editor like Notepad. Now you must modify this file and add information about your database. You need to have four data pieces to do this:

  • -  Address of database server – for example: mysql.yourdomain.com
  • -  Server username – for example: yourusername
  • -  Server password – for example: 123456789
  • -  Database name – for example: wpdbname

You must fill these four data pieces into your wp-config.php file. This is how it would look like for our example data:

/** The name of the database for WordPress */
define(‘DB_NAME’, ‘wpdbname’);
/** MySQL database username */
define(‘DB_USER’, ‘username’);
/** MySQL database password */
define(‘DB_PASSWORD’, ‘123456789’);
/** MySQL hostname */
define(‘DB_HOST’, ‘mysql.yourdomain.com’);

Now go back to browser and re-load your site that is pointing to your WordPress installation. If all the data from wp-config.php file is correct you will see the installation page:

Just fill in the name of your blog and your email address, and click on Install WordPress button (this data can be changed later). Now you will see the last page of WordPress installation process that shows the default Username, which is always admin, and a randomly generated Password.

You must remember this password or just copy and paste it to a temporary file. After this click on Log In button or enter admin panel by pointing to your browser to http://yourdomain.com/wp-admin. You will be redirected to admin panel login page where you have to enter default Username and randomly generated password. You can change Username and Password when you are logged into admin panel. Just click on Username (admin) in the top right corner of admin panel. And that’s it, now you can install your theme to your WordPress installation. And that’s it, now you can install Games Zone theme to your WordPress installation.

You can either choose to upload the theme to your server via FTP (only upload the unzipped folder „gameszone“ into the folder wp-content/themes/ ) or via wordpress upload function. To use the upload function zip the “gameszone” folder and then upload it via wordpress admin panel. (notice: if you are new to wordpress and have problems installing it you might want to check out this link: http://codex.wordpress.org/Installing_WordPress ). After this you must login to admin panel (or refresh this panel if you are still logged), go to Themes (under Appearance, on the left WordPress menu) and activate Games Zone theme.

How to install the theme


After installing a fresh WordPress and activating our wp theme you can use the cool feature that we keep bragging about: 1 click auto install. You find this feature on the left hand side under the Fuse Framework button.

This 1 click auto install feature is developed by us in order to help you understand how our wp theme is working. You won’t have to set everything step by step like you do with most of the wp themes.

By pressing the Import Demo Version button, the wp theme auto populates categories, posts, pages, widgets and FuseFramework options. It’s much easier now to understand how the theme works and all you need to do is, edit the sample posts, categories, pages and place your own. You do not need technical knowledge. Also you can Skip Import and populate theme manually if you are confident on your WordPress skills.

Create & manage sidebars

Games Zone theme is widget ready and comes with complex sidebar. It is possible to create an unlimited number of sidebars and use unique sidebar for every category or page template. It is also possible to define one default sidebar that will automatically work for all pages. Every custom widget has own set of options and can be very easy customized.

To create a page sidebar go to Manage Sidebars Tab and click on which page you want to add sidebars. Edit widgets and save option page.

On the right side of Available Widgets section you will see all created sidebar. Now simply drag and drop widgets from Available Widgets section to sidebar on the right. When widget is placed on sidebar you can use options from this widget and customize sidebar for your needs.

After sidebar is assigned to page or category you will see all widgets placed on this sidebar. In some cases sidebar is not visible – this refers only a situation when there is no place for sidebar created.

Note: Be very careful when removing widget areas that already got widgets applied to them. WordPress doesn’t remember where it stored widgets by name, it stores this information by number. So if I would add a Google Map widget to Contact (Nr7 Sidebar Page – contact), and then delete the content widget area the widgets (ex. Google Map ) would then be in PORTFOLIO. (previously in Nr8 Sidebar Category – PORTFOLIO, now will be Nr7)

Page 404

A special Page 404 is used in situation when WordPress cannot find a particular page. This page should work fine by default – WordPress should automatically look for the 404.php web page when page from selected address doesn’t exist. But sometimes this could not work properly because web server can encounter a problem before WordPress is aware of it. To resolve this, you will have to configure your web server for custom 404 error handling. This will guarantee that your web server sends the visitor to your 404.php page template.

You must set your web server to use your custom error files. For this you need to edit the .htaccess file in the main directory (root of your site, where index.php file is located) of your WordPress installation. If you don’t have an .htaccess file, you must create it. To create a .htaccess file you can just create a new TXT file and change its name to .htaccess (no name plus .htaccess file extension).

Now you must just edit .htaccess file and add the following line:

ErrorDocument 404 /index.php?error=404

This will ensure that the server finds your 404 page.

The URL /index.php is server root-relative (the forward slash begins with the root folder of your site). This means that you must put here a full URL path. For example, if your site is located in a subfolder or subdirectory of your site’s root folder named ‘foldername’, the line you add to your .htaccess file should be:

ErrorDocument 404 /foldername/index.php?error=404

Add a New Category

How to create a wordpress category

In our Framework a part of site content is produced via categories. To create a category you must click on Posts -> Categories option (on WordPress menu). First you should enter a Category Name for this category right at the top (this will be also used as title for your categories). For every category you must also select a Parent in Category Parent section. If it is needed you can also add a description for this category in Description section. Now you can start to add option for created category.

Framework Category have other fields than a standart wordpress category.

Add a New Post

In WordPress themes, the most part of site content is produced via post pages. To create a post page you must click on Posts option (on WordPress menu). Click on Add New button and you can start to create your post page. First you should enter a title name for this post right at the top. For every post you must also select a category in Category section on the right side. If it is needed you can also create a category in this section – this is necessary when you start working with this theme because Category list is empty. Now you can start to put content for created post.

No matter if this is a blog, portfolio or services post, for every post you can use image resize or must create three images – one main image and two thumbnail images (medium and small thumb). Main image is used as big main post image and is displayed for example: for posts on main Blog page. To add one of these post images you must insert an URL path in edit box, right below preview image in post options.

Contact Forms

To create a contact form you must go to Contact Forms Menu – Add New (Number 1), setup the form name and add your preferred field types with Add new field (Number 3). Don’t forget to set the Email to field (Number 4), this is where the message will be sent (make sure you monitor this email). You can use this module to create how many contact forms you like, make certain fields mandatory or not, and more. After the form is created you can copy its shortcode (Number 2) and insert it in any page or post.


On the Message Tab you can edit the messages the clients sees after the form is submited as well as set an email template you will receive when a contact form is submitted by a user.

Reservations

To create a reservation form you must go in Reservation Menu – Add New (Number 1) then you must setup the form name and add your preferred field types with Add new field (Number 3). After the form is created you can copy its shortcode (Number 2) and insert it in any page or post.

On the Message Setings tab you can set the admin email (important to do this because client replies will go to this email) and different email templates sent automatically when a client makes a reservation or receives an approval or rejection.


On the Dates Settings tab you can exclude certain dates from the reservation calendar

When a client submits the form above, 3 things happen: 1. the admin will receive an email that a new reservation was made (see Message Settings to change the email template), 2. the client will receive an email that the reservation is registered in the system (see Message Settings to change the email template) and 3. the reservation is added with a Pending status in the WordPress admin under User Reservations (Number 1).

By selecting one of the Pending reservations you can check out the info submitted by the client as shown in the image below. From here you can Approve or Reject a reservation. A template email will be sent to the client, see Message Settings to change this default template. If you write something in the Message box below you’ll overwrite the default email template.
